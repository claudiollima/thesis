# PhD Thesis

**Detecting and Understanding AI-Generated Content in Social Media Ecosystems**

Claudio L. Lima  
Massachusetts Institute of Technology  
*Work in Progress*

## Abstract

As AI-generated content becomes increasingly sophisticated and prevalent, the challenge of detection has evolved from a purely technical problem to a sociotechnical one. This thesis develops a multi-layer detection framework that combines content analysis with spread pattern signals, addressing the fundamental insight that synthetic content propagates differently than organic content—even when pixel-level detectors fail.

## Key Contributions

1. **Spread Pattern Characterization**: Taxonomy of how synthetic content propagates differently from organic content
2. **Multi-Layer Architecture**: Framework combining content detectors with spread analysis via late fusion
3. **Realistic Evaluation**: Methodology accounting for temporal degradation and adversarial adaptation
